##  Project Overview

This project presents an interactive **Sales Analytics Dashboard** built using **Power BI**. It provides insights into sales performance, profit trends, regional distribution, and product-level analysis to support data-driven decision-making.


##  Objectives

* Analyze overall sales performance and profitability
* Identify top-performing products and regions
* Track monthly and quarterly sales trends
* Monitor return rates and key KPIs
* Provide an interactive dashboard for business insights


* Contains data model, visuals, and dashboard design


##  Dashboard Features

###  Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Total Units Sold
* Total Returns

###  Visualizations Included

* Sales by Quarter
* Sales by Region
* Sales Trend by Month
* Return Percentage by Region
* Top 5 Products by Sales
* Category-wise performance (Accessories, Binders, Bookcases, etc.)

##  Insights Derived

* Highest sales recorded in **Q4**, indicating strong year-end performance
* South region** contributes the most to overall revenue
* Sales show a **declining trend across months** after peak period
* Top products drive a significant portion of total revenue
* Return rates are relatively low, indicating efficient operations
